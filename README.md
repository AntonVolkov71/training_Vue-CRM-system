# CRM System

Приложение создано для детального разбора фреймворка Vue.js :
* регистрация пользователя с валидацией
* создание роутов и работа с ними
* использование плагинов, фильтров и миксинов
* локализцаия сайта
* SEO
* использован хостинг и БД Firebase
* работа с Material CSS
